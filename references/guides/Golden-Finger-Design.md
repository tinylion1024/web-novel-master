# Golden Finger Design Guide

The Golden Finger is the protagonist's cheat/external ability, the core tool that drives the plot and excites readers.

---

## 1. Definition of Golden Finger

**Golden Finger = The protagonist's unique special ability/advantage**

It is the key that makes the protagonist exceed ordinary people, and the core element that attracts readers to web novels.

| Feature | Description |
|---------|-------------|
| **Uniqueness** | Only the protagonist has it, other characters don't |
| **Growth** | Can upgrade/evolve as the story progresses |
| **Limitations** | Must have usage limits to maintain balance |
| **Extensibility** | Can extend to various domains, generating plots |

---

## 2. Golden Finger Type System

### 1. System Flow

The protagonist obtains a system that can issue tasks and give rewards.

```
【Core Mechanism】
Issue task → Complete → Reward → Strength increase
        ↓
Side quest → Special reward
        ↓
Hidden quest → Ultimate reward
```

| Type | Feature | Example |
|------|---------|---------|
| Task System | Issue tasks, complete for rewards | "Ding! Complete task X, reward Y" |
| Choice System | Branching choices, different results | "Ding! Please choose: A/B/C" |
| Check-in System | Daily check-in, cumulative rewards | "Sign in for 7 days, reward..." |
| Synthesis System | Synthesize low-level to high-level | "Synthesis successful! Obtained artifact!" |
| Live Stream System | Display abilities for rewards | "Audience gifts convert to abilities" |

**System Limitation Design:**
```
- Daily task limit
- Punishment mechanism (task failure penalty)
- Cooldown time (skills can't be used consecutively)
- Upgrade conditions (system itself needs growth)
```

### 2. Rebirth Memory Flow

The protagonist returns to the past with memories of a previous life.

```
【Core Mechanism】
Previous life memory → Prophet advantage → Change fate
        ↓
Know secrets/history/future
        ↓
Advance planning/avoid risk/revenge on enemies
```

| Type | Feature | Example |
|------|---------|---------|
| Apocalypse Rebirth | Return with apocalypse memories | Knows zombie outbreak timing |
| Immortal Emperor Rebirth | Rebirth after reaching cultivation pinnacle | Knows all technique secrets |
| Stone Gambling Rebirth | Rebirth before stone gambling | Knows each stone's value |
| Campus Rebirth | Return to student era | Knows who will betray |

**Rebirth Memory Limitations:**
```
- Memory incomplete (only remembers important things)
- Butterfly effect (changing one thing affects future)
- Enemies stronger (rebirth doesn't mean invincibility)
```

### 3. Inheritance Flow

The protagonist receives inheritance from ancient/mythical powerhouses.

```
【Core Mechanism】
Discover inheritance → Accept trial → Receive inheritance
        ↓
Inheritance layered
        ↓
Gradually unlock higher-level content
```

| Type | Feature | Example |
|------|---------|---------|
| Technique Inheritance | Obtain lost techniques | Ancient divine arts, forbidden secrets |
| Memory Inheritance | Receive powerhouse memories | Thousand years of experience, dan master |
| Skill Inheritance | Obtain special skills | Pill refining, artifact making, formations |
| Master Disciple Inheritance | Accept master-disciple transmission | Become successor |

**Inheritance Limitation Design:**
```
- Need to pass trial to unlock next layer
- Inheritance has usage conditions (cultivation/body/blood)
- Inheritance incomplete, need to supplement
- After accepting, monitored by original owner/have conditions
```

### 4. Special Ability Flow

The protagonist obtains some kind of supernatural ability.

```
【Core Mechanism】
Ability awakening → Ability growth → Application expansion
        ↓
Discover deeper layers of the ability
        ↓
Ability gets stronger
```

| Type | Feature | Example |
|------|---------|---------|
| Perspective | See through everything | Stone gambling, medical, appraisal |
| Mind Reading | Read thoughts | Negotiation, interrogation, flirting |
| Precognition | See the future | Investment, risk avoidance, gaming |
| Repair | Repair anything | Medical, artifact making, talisman |
| Summoning | Summon beings/items | Summon pets, army |

**Special Ability Growth Design:**
```
Ability levels: A→B→A→S→SS→SSS
Growth conditions: Usage accumulation/task completion/specific triggers
Special forms: Ability evolution/fusion/mutation
```

### 5. Storage Space Flow

The protagonist has an independent space.

```
【Core Mechanism】
Obtain space → Store/plant/breed
        ↓
Space upgrade (bigger/faster/has spiritual energy)
        ↓
Space output → Protagonist advantage
```

| Type | Feature | Example |
|------|---------|---------|
| Storage Space | Storage only | Store supplies/living things |
| Planting Space | Can plant | Spiritual medicine/accelerated growth |
| Breeding Space | Can breed | Rare pets/livestock |
| Mixed Space | Planting + breeding + dwelling | Self-sufficient micro world |

**Space Limitation Design:**
```
- Space capacity limit
- Entry/exit count limit
- Space upgrade conditions (need rare materials)
- Can't let outsiders know
```

### 6. Bloodline/Constitution Flow

The protagonist has a special bloodline or constitution.

```
【Core Mechanism】
Bloodline awakening → Inherit power → Bloodline evolution
        ↓
Understand bloodline history
        ↓
Bloodline purer, power stronger
```

| Type | Feature | Example |
|------|---------|---------|
| Dragon Bloodline | Powerful body | Breathe fire/fly/scales |
| Phoenix Bloodline | Nirvana rebirth | Fire/rebirth/phoenix true fire |
| Shennong Bloodline | Medical talent | Taste all herbs/detoxify/refine |
| Assassin Bloodline | Combat talent | Silent/instant kill |

**Bloodline Limitation Design:**
```
- Bloodline awakening needs conditions
- Bloodline has side effects (berserk/lose control)
- Bloodline purity affects power
- Bloodline has inheritance history (may have original owner's will)
```

## 3. Golden Finger Design Principles

### 1. Uniqueness Principle

The Golden Finger must be unique to the protagonist, not common.

**Anti-example:**
> "Ding! System activated!"—system novels are everywhere

**Improvement:**
> System + unique mechanism: e.g., "only host can see hidden information" + "system tasks deeply bound with plot"

### 2. Limitation Principle

The Golden Finger must have usage limitations, otherwise there's no point.

**Anti-example:**
> Unlimited system, can get whatever reward you want

**Improvement:**
> Daily task limit + punishment mechanism + upgrade conditions

### 3. Growth Principle

The Golden Finger must be able to grow, not be maxed from the start.

**Anti-example:**
> Level 99 from Chapter 1, no future看点

**Improvement:**
> Golden Finger needs cultivation/upgrade/evolution, unlock gradually

### 4. Balance Principle

The Golden Finger cannot be too overpowered, otherwise there's no tension.

**Anti-example:**
> Has system, can solve anything, protagonist never faces real crisis

**Improvement:**
> System can only assist, truly powerful still requires protagonist's effort

### 5. Extensibility Principle

The Golden Finger must extend to different domains, generating diverse plots.

**Anti-example:**
> Can only be used for combat, too single-dimensional

**Improvement:**
> Perspective can be used for: medical, stone gambling, appraisal, flirting (multiple scenarios)

## 4. Golden Finger and Plot Integration

### Early Stage (Chapter 1-10)

Golden Finger just obtained, mainly about understanding and familiarizing.

```
Chapter 1: Obtain Golden Finger
Chapter 2-3: Initial use, experience ability
Chapter 4-5: Encounter first challenge
Chapter 6-10: Use Golden Finger to solve, obtain first big爽点
```

### Middle Stage (Chapter 11-50)

Golden Finger becomes protagonist's important advantage, generates more plots.

```
- Golden Finger upgrades, unlocks new abilities
- Enemies start targeting protagonist's Golden Finger
- Troubles/chasers brought by Golden Finger
- Golden Finger and romance line intertwined
```

### Late Stage (Chapter 51+)

Golden Finger mature, protagonist and Golden Finger become one.

```
- Golden Finger ultimate form
- Golden Finger origin/history revealed
- Golden Finger and final boss relationship
- Golden Finger's price/hidden attributes
```

## 5. Golden Finger Design Checklist

### Before Design

- [ ] What is the Golden Finger type?
- [ ] What is the Golden Finger's core mechanism?
- [ ] What are the Golden Finger's limitations?
- [ ] How does the Golden Finger grow?
- [ ] What troubles does the Golden Finger bring?

### After Design

- [ ] Is the Golden Finger unique? (Not common)
- [ ] Are limitations reasonable? (Too weak is meaningless, too strong has no看点)
- [ ] Is the growth line clear? (Can support how many chapters?)
- [ ] Can it extend to multiple domains? (Avoid single-dimensional)

## 6. Golden Finger and Excitement Point Coordination

### System Flow + Slap Face爽点

```
System issues task: "Slap face those who look down on you"
        ↓
Complete task
        ↓
Obtain reward + Slap face success
        ↓
Double excitement
```

### Rebirth Flow + Revenge爽点

```
Previous life memory: "Who harmed me back then"
        ↓
Return, collect evidence
        ↓
Revenge success + Take back everything
        ↓
Strong excitement
```

### Inheritance Flow + Leveling Up爽点

```
Obtain first layer of inheritance
        ↓
Cultivate/pass trial
        ↓
Unlock second layer, power surges
        ↓
Leveling up excitement
```

### Special Ability Flow + Show-off爽点

```
Use ability: Display publicly
        ↓
Crush opponent
        ↓
Everyone shocked
        ↓
Show-off success
```
