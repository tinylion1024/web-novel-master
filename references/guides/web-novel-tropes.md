# 爆款网文套路大全

网文黄金套路是经过市场验证的写作模板。掌握这些套路，能让你的小说自带流量属性。

---

## 一、退婚流

### 套路概述

经典废材逆袭模板，主角前期被打压，后期碾压打脸。

### 核心结构

```
【第一阶段：羞辱】
→ 未婚妻/家族当众退婚
→ 主角被嘲讽"废物"
→ 羞辱足够狠，读者憋一口气

【第二阶段：蓄力】
→ 主角意外获得传承/系统/记忆
→ 实力开始增长
→ 暗中发育，不张扬

【第三阶段：打脸】
→ 与羞辱自己的人重逢
→ 身份/实力完全碾压
→ 金句收尾："当年你说的那句话，还记得吗？"
```

### 变形玩法

| 类型 | 羞辱方式 | 打脸方式 |
|-----|---------|---------|
| 退婚流 | 当众退婚，扔休书 | 身份揭露，未婚妻后悔 |
| 休夫流 | 女强男弱，反向打脸 | 女主发现男主是隐藏大佬 |
| 叛族流 | 被家族除名 | 回来后发现家族需要他 |

### 名句模板

> "今日退婚，此后男婚女嫁，各不相干。"
>
> "三十年河东，三十年河西，莫欺少年穷！"

## 二、系统流

### 套路概述

主角获得外挂系统，完成任务即可获得奖励，节奏明快。

### 核心结构

```
【激活】
→ 绝境/死亡/重大挫折
→ 系统激活
→ 新手任务发布

【任务驱动】
→ 任务完成 → 奖励 → 实力提升
→ 支线任务 → 特殊奖励
→ 隐藏任务 → 高级奖励

【升级循环】
→ 变强 → 新敌人 → 更强任务 → 更强奖励
```

### 系统类型

| 类型 | 特点 | 爽点 |
|-----|------|------|
| 任务系统 | 完成发布任务获得奖励 | 任务奖励感 |
| 选择系统 | 选项分支，不同选择不同结果 | 命运掌控感 |
| 签到系统 | 每日签到，累积奖励 | 习惯依赖 |
| 召唤系统 | 可召唤历史人物助战 | 名将对决 |
| 合成系统 | 低级物品合成高级 | 炼器快感 |
| 直播系统 | 直播获得打赏/能力 | 观众互动 |

### 名句模板

> "叮！检测到宿主生命垂危，是否激活新手大礼包？"
>
> "叮！恭喜宿主完成首杀，奖励屠龙刀一把！"
>
> "系统在手，天下我有。"

## 三、重生流

### 套路概述

主角带着前世记忆回到过去，利用先知优势逆天改命。

### 核心结构

```
【死亡/重大失败】
→ 主角遭遇背叛/失败/死亡
→ 临死前记住关键信息

【重生节点】
→ 回到过去某个时间点
→ 保留前世记忆和技能
→ 知道未来即将发生的大事

【先知优势】
→ 提前布局，规避风险
→ 抱紧大腿，投资潜力股
→ 报复仇人，碾压对手
```

### 重生类型

| 类型 | 特点 | 核心爽点 |
|-----|------|---------|
| 末世重生 | 丧尸/灾难前重生 | 囤货/建造基地/杀异变兽 |
| 校园重生 | 回到学生时代 | 弥补遗憾/追回初恋/考名校 |
| 历史重生 | 穿回古代/民国 | 已知历史走向/科技抄作业 |
| 仙帝重生 | 修炼到极致后重生 | 法宝/功法/丹药全知道 |
| 赌石重生 | 重生到赌石前 | 一刀富/翡翠帝王绿 |

### 名句模板

> "这一世，我不会再让你们得逞。"
>
> "前世我瞎了眼，这辈子......该我负天下人了。"

## 四、升级流

### 套路概述

主角在一个明确的等级体系中不断突破，节奏感强。

### 等级体系示例

```
【修炼等级】（玄幻/仙侠）
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 大乘 → 渡劫 → 真仙

【都市等级】（异能/商战）
普通人 → 入门 → 精英 → 大师 → 宗师 → 传说 → 神级

【学院等级】（校园/异界）
青铜 → 白银 → 黄金 → 铂金 → 钻石 → 王者 → 荣耀
```

### 升级节奏公式

```
每10章：小突破1次
每20章：中突破1次
每50章：大突破1次
瓶颈 → 危机 → 顿悟 → 突破 → 新境界
```

### 升级名场面

- 突破时的天地异象
- 敌人发现主角突破时的震惊
- 突破后的实力展示

## 五、切片流

### 套路概述

主角分出多个分身/马甲，每个身份都有独立故事线。

### 核心结构

```
【本体】实力强，隐藏身份
【分身/马甲】各自有独立身份
【身份切换】根据情况切换
【最终揭示】所有马甲都是同一人
```

### 切片类型

| 类型 | 特点 | 爽点 |
|-----|------|------|
| 分身流 | 多个分身同时行动 | 多线操作/分身反杀 |
| 马甲流 | 不同场合不同身份 | 身份切换/揭露震撼 |
| 人格分裂 | 主人格/副人格切换 | 不同人格不同能力 |
| 替身流 | 多个替身代替本人 | 狡兔三窟/信息差 |

### 名场面

> "其实......三个身份都是他。"
>
> "什么？神医、兵王、首富......都是同一个人？"

## 六、直播流

### 套路概述

主角在直播平台展示能力，观众互动影响剧情。

### 核心结构

```
【直播开始】
→ 展示特殊能力/技艺
→ 观众震惊/质疑
→ 刷礼物/弹幕互动

【打脸环节】
→ 有人质疑是假的
→ 主角用实力证明
→ 观众刷屏"牛X"

【热度提升】
→ 粉丝增加/打赏爆发
→ 引发更大关注
→ 连麦/PK其他主播
```

### 直播类型

| 类型 | 内容 | 爽点 |
|-----|------|------|
| 荒野直播 | 荒野求生/建造 | 硬核技能/化险为夷 |
| 游戏直播 | 电竞/通关 | 技术碾压/五杀超神 |
| 玄学直播 | 算命/风水/捉鬼 | 算准了/鬼出现 |
| 美食直播 | 顶级厨艺/网红店 | 好吃到哭/排队三公里 |
| 鉴宝直播 | 古董鉴定/赌石 | 一夜暴富/打眼 |

## 七、宠兽/宠物流

### 套路概述

主角拥有独特的宠物/宠兽，共同成长。

### 核心结构

```
【获得宠物】
→ 意外获得稀有宠物
→ 宠物看似普通，实则强大
→ 签订契约，共同成长

【共同成长】
→ 宠物进化/升级
→ 宠物救主/主救宠物
→ 羁绊加深，实力倍增

【终极形态】
→ 宠物觉醒终极形态
→ 横扫一切，震撼全场
→羁绊无敌
```

### 宠物类型

| 类型 | 示例 | 特点 |
|-----|------|------|
| 进化型 | 小蛇→蛟龙→真龙 | 成长线清晰 |
| 合体型 | 宠物与主人合体 | 战力倍增 |
| 召唤型 | 可召唤多只宠物 | 群战优势 |
| 共生型 | 宠物即主人 | 命悬一线 |

## 八、传承流

### 套路概述

主角获得远古/上古大能传承，一步步解锁。

### 核心结构

```
【发现传承】
→ 意外获得传承之地
→ 接受传承考验
→ 通过考验，获得传承

【解锁升级】
→ 传承分多层次
→ 逐步解锁更高级内容
→ 每次解锁都是爽点

【终极传承】
→ 获得最强传承
→ 实力碾压同辈
→ 成为新的传奇
```

### 传承类型

| 类型 | 示例 | 爽点 |
|-----|------|------|
| 功法传承 | 获得上古功法 | 功法碾压 |
| 技能传承 | 获得失传技能 | 技艺震撼 |
| 记忆传承 | 获得大能记忆 | 全知全能 |
| 衣钵传承 | 接受师徒传承 | 身份逆转 |

## 九、组合套路公式

爆款网文通常是多个套路的组合。

### 常见黄金组合

| 组合 | 说明 | 适用类型 |
|-----|------|---------|
| 系统+升级 | 系统发任务，奖励促升级 | 玄幻/都市 |
| 重生+复仇 | 重生后报复仇人 | 都市/历史 |
| 退婚+系统 | 退婚激活系统 | 玄幻/修仙 |
| 直播+打脸 | 直播打脸质疑者 | 都市/异能 |
| 传承+升级 | 传承辅助升级 | 玄幻/仙侠 |
| 重生+甜宠 | 重生追回爱人 | 都市/校园 |
| 系统+甜宠 | 系统撮合姻缘 | 都市/现代 |
| 马甲+身份揭示 | 多马甲最终揭开 | 都市/职场 |

### 套路选择原则

1. **选一个主套路** 作为核心框架
2. **加一个副套路** 增加变化和爽点
3. **情感线贯穿** 增加读者粘性
4. **反派持续搞事** 保持冲突不断

## 十、套路避坑指南

| 坑 | 问题 | 解决 |
|---|-----|------|
| 套路太旧 | 读者审美疲劳 | 旧瓶新酒，加入新元素 |
| 套路太杂 | 没有主线 | 一个主套路贯穿始终 |
| 节奏失控 | 前10章没爽点 | 第1章必须有钩子 |
| 反派太蠢 | 打脸不够爽 | 反派要强，打脸才爽 |
| 系统太无敌 | 没有危机感 | 给系统加限制/冷却 |
| 升级太容易 | 没有成就感 | 瓶颈+危机+突破循环 |
