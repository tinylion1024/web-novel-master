# Phase 7 – Polish & Pacing

**目标**：深度润色、去除AI味、节奏调整

---

## 7.1 AI味清除

AI 生成文本有明显语言模式，必须逐项消除。

### 高频AI词汇（出现即改）

| AI词汇 | 替换方案 |
|--------|---------|
| "值得注意的是" | 直接删除或改写 |
| "不禁""油然而生" | 用具体情绪替代 |
| "仿佛""宛如" | 用真实比喻或删除 |
| "这一刻" | 删除或具体化 |
| "彰显""诠释""赋能" | 用具体动作替代 |

### 句式问题

| AI句式 | 问题 | 修正 |
|--------|------|------|
| "他做了A，他做了B，他做了C" | 句式单调 | 合并、省略主语 |
| "她很美丽，她很善良，她很温柔" | 形容词堆砌 | 用场景展示 |
| "他深吸一口气，然后…" | 流水账 | 直接进入情节 |

### 的字控制

- 每句话不超过 2 个"的"
- 禁止出现"的的"
- 修饰能简就简

## 7.2 语言质量提升

### 白描手法

用最少的笔墨精确捕捉本质。

```
反例：璀璨的星空下，瑰丽的晚霞映照着静谧的湖面
正例：湖面映着天光，水波不兴。远处有人点了一盏灯。
```

### 留白手法

不说的部分比说的更有力量。

```
正例：
他看着她的背影消失在安检口。
口袋里的手机响了。
他没接。
```

### 情绪外放

网文情绪要直白，不要过于含蓄。

```
反例：她心中有些难过
正例：眼眶红了，她狠狠吸了口气，把眼泪逼回去
```

## 7.3 节奏调整

### 字数控制

| 章节类型 | 字数范围 |
|---------|---------|
| 标准章节 | 2000-2500字 |
| 高潮章节 | 2500-3000字 |
| 过渡章节 | 1800-2500字 |

### 节奏检查

- [ ] 每500字必须有情节推进或角色成长
- [ ] 憋屈段落不超过3段，必须有释放
- [ ] 爽点描写要"够狠"，不能含蓄

### 憋爽比例

```
轻松期（日常/甜蜜）：爽7 : 憋3
压迫期（被压制/困境）：憋6 : 爽4
爆发期（逆袭/打脸/高潮）：爽8 : 憋2
```

---

## 7.4 人物语气检查

- [ ] 对话符合人物性格
- [ ] 高冷角色不应话多
- [ ] 幽默角色应有吐槽
- [ ] 没有废话对话（"你好""吃了吗"）

---

## 7.5 逐章润色流程

对每一章执行：

1. **AI味清除** — 逐项检查高频词、句式、的字密度
2. **语言质量** — 运用白描、留白、情绪外放
3. **节奏检查** — 对照憋爽比例，调整节奏
4. **人物语气** — 检查对话是否符合性格
5. **字数调整** — 不足扩充，过多精简
6. **更新 JSON** — 润色后标记 `polished: true`
