# Data Directory

市场数据目录 - 题材分析、爆款数据、平台特点。

## 目录结构

```
Data/
├── Market/                # 市场分析
│   ├── Platform-Rank.md  # 平台排行榜分析
│   ├── Trend-Analysis.md  # 市场趋势分析
│   └── Reader-Demographic.md # 读者画像
├── Genre/                 # 题材数据
│   ├── Urban-Stats.md    # 都市题材数据
│   ├── Xianxia-Stats.md  # 玄幻题材数据
│   ├── Romance-Stats.md  # 言情题材数据
│   └── Mystery-Stats.md # 悬疑题材数据
├── Hit-Analysis/          # 爆款分析
│   ├── Top-Performers.md # 爆款案例分析
│   ├── Growth-Patterns.md # 增长模式
│   └── Reader-Feedback.md # 读者反馈数据
└── Platform/             # 平台特点
    ├── Qidian.md         # 起点中文网
    ├── Fanqie.md        # 番茄小说
    ├── Feilu.md         # 飞卢小说
    └── Qimao.md          # 七猫小说
```
