# 都市重生最小项目示例

这是一个可公开复用的虚构样例，展示 `manage_novel_project.py init` 之后，作者如何把基础模板补成可继续写作的项目。

## 输入

```text
类型：都市重生
核心设定：主角能从旧物中读取关键记忆，但每次使用都会失去一段自己的往事。
目标：用 3 章建立债务危机、能力限制和首个反转。
```
