# 数据目录

市场资料目录 - 题材研究假设、平台观察与来源核验记录。

## 目录结构

```
Data/
├── Market-Data.md          # 待核验市场资料与研究假设
└── Source-Registry.md      # 原始来源、日期、口径与核验状态
```

## 内容概览

| 文件 | 内容 | 使用方式 |
|------|------|---------|
| **Market-Data.md** | 行业、用户、题材与平台相关资料 | 仅作为选题研究假设；引用前回查原始来源 |
| **Source-Registry.md** | 每项资料的来源、日期、统计口径和核验状态 | 先登记、后引用；无原始来源的数据不得作为事实使用 |

## 使用规则

1. 先在 [数据来源核验登记表](./Source-Registry.md) 记录原始链接、日期和统计口径。
2. 未完成核验的数字只能作为内部研究线索，不得用于 README、商业决策或对外陈述。
3. 平台规则、榜单和市场趋势变化快；提交作品前应以平台官方最新规则为准。
