## Q9: 读者定位

```
Question: 目标读者是？
Options:
- 【小白读者】喜欢直白爽文，不需要烧脑
- 【老白读者】喜欢有深度/有脑子的爽文
- 【女性读者】偏甜宠/感情线为主
- 【男性读者】偏打脸/升级/战斗
- 【大众读者】男女通吃，老少皆宜
- 🎲 随机生成
```

**记录**：`targetAudience`

---

## Q10: 章节数量

```
Question: 你计划创作多少章？
Options:
- 20章（短篇，约5-6万字）⭐
- 30章（中篇，约7-10万字）
- 50章（中长篇，约12-18万字）
- 100章+（长篇，20万字+）
- 自定义
```

> **偏好集成**：如 `typicalChapterCount` 有值，标记对应选项为⭐

**记录**：`chapterCount`

---

## Q11: 特殊要求（可选）

```
Question: 有没有特殊要求？（均可跳过）
Options:
- 有必须包含的情节/场景（请描述）
- 有绝对不能出现的内容（请描述）
- 字数偏好（默认每章2000-3000字）
- 没有特殊要求，按标准来
```

**记录**：`specialRequirements`
