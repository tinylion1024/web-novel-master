# 爽点设计指南

爽点是网文的核心驱动力。读者追更的本质是"等爽"——等主角打脸、等逆袭、等发糖。

---

## 一、爽点的本质

**爽点 = 压抑后的释放**

没有压抑，就没有爽感。爽点的前置条件是"憋"——主角被羞辱、被压制、被看不起，然后反转。

```
压抑（憋） → 释放（爽）
    ↓           ↓
  越憋越足    越放越爽
```

**反面教材：**
主角突然变强，没有任何铺垫，读者只会觉得"哦"。

**正面教材：**
被退婚、被嘲笑、被妹妹看不起......三章后主角亮出身份，读者才爽。

---

## 二、爽点类型体系

### 1. 打脸爽点

**前置条件：** 必须有足够狠的羞辱

| 羞辱阶段 | 打脸阶段 | 效果 |
|---------|---------|------|
| 轻度羞辱 | 小打脸 | 微爽 |
| 中度羞辱（当众） | 中打脸（实力碾压） | 大爽 |
| 重度羞辱（涉及亲人） | 极致打脸（团灭） | 爆爽 |

**打脸爽点公式：**
```
【1】羞辱够狠：废物/垃圾/不配/滚
【2】时机成熟：主角微微一笑
【3】实力展示：一个动作/一句话
【4】全场震惊：死寂/倒吸凉气/脸色惨白
【5】金句收尾：一句让读者记住的话
```

### 2. 升级爽点

**前置条件：** 等级体系明确，读者能看到成长

| 升级类型 | 示例 | 爽点 |
|---------|------|------|
| 突破型 | 筑基→金丹，天降异象 | 仪式感 |
| 击杀型 | 越阶杀敌，以弱胜强 | 成就感 |
| 获得型 | 获得神器/功法/宠物 | 满足感 |
| 解锁型 | 传承解锁，功法升级 | 期待感 |

**升级爽点公式：**
```
【1】困境：敌人强大，无法战胜
【2】契机：意外获得/顿悟/触发
【3】突破：实力暴涨，等级提升
【4】验证：用新实力打败旧敌人
【5】震撼：旁观者的反应
```

### 3. 情感爽点

**前置条件：** 读者已经对角色有情感投入

| 情感爽点 | 触发条件 | 效果 |
|---------|---------|------|
| 吃醋爽 | 看到对象和别人亲密 | 占有欲 |
| 护短爽 | 霸气护住自己人 | 安全感 |
| 告白爽 | 终于表白/被表白 | 甜蜜感 |
| 撒糖爽 | 甜蜜互动/壁咚/公主抱 | 姨母笑 |
| 复仇爽 | 报复伤害过主角的人 | 快感 |

**情感爽点公式：**
```
【1】误解/危机：两人关系出现波折
【2】真情流露：关键时刻表现真情
【3】名场面：壁咚/强吻/霸气宣言
【4】金句：让读者想截图的话
【5】余韵：甜甜的/暖暖的/心动
```

### 4. 装逼爽点

**前置条件：** 主角有实力但一直隐藏

| 装逼类型 | 示例 | 爽点 |
|---------|------|------|
| 低调装逼 | 不想出手，被迫展示 | 无奈感 |
| 身份装逼 | 亮出隐藏身份 | 震撼感 |
| 财富装逼 | 随手消费千万/亿万 | 满足感 |
| 势力装逼 | 身后大佬/小弟/势力 | 威慑感 |

**装逼爽点公式：**
```
【1】被打压：被嘲讽穷/弱/没背景
【2】被逼迫：被要求证明/出手/离开
【3】无奈叹气：既然你们非要看...
【4】展示实力：轻松碾压/随手就是王炸
【5】金句：不针对谁，在座都是...
```

## 三、爽点密度设计

### 每2000字的爽点配置

| 爽点类型 | 数量 | 位置 | 强度 |
|---------|------|------|------|
| 小爽点 | 1-2个 | 均匀分布 | 轻 |
| 中爽点 | 1个 | 章中/章末 | 中 |
| 大爽点 | 1个 | 章节高潮 | 强 |
| 钩子 | 1-2个 | 章末 | 引流 |

### 节奏公式

```
爽 → 憋 → 更爽 → 大憋 → 大爽
↓    ↓    ↓      ↓     ↓
日常 压迫  小爽  绝境   逆袭
```

### 各类型章节爽点配置

**第1章（导入）：**
- 必须有觉醒爽点（系统激活/重生/退婚打脸）
- 让读者立刻知道"这文很爽"

**第3章（验证）：**
- 必须有首次打脸爽点
- 验证"爽文"标签，建立读者预期

**高潮章（10/20/30章）：**
- 大爽点密集
- 多重爽点叠加
- 金句轰炸

**过渡章（5/15/25章）：**
- 小爽点维持节奏
- 情感铺垫/关系推进
- 为下一波爽点蓄力

## 四、爽点设计检查清单

### 写前检查

- [ ] 本章爽点类型是什么？
- [ ] 爽点前置的"憋"够不够？
- [ ] 爽点强度够不够？
- [ ] 有没有让读者想截图的金句？

### 写后检查

- [ ] 爽点出现时机对不对？（太早/太晚都不行）
- [ ] 爽点之后有没有余韵？
- [ ] 爽点密度是否合适？（太少水，太多腻）
- [ ] 和上章爽点有没有重复？（要有变化）

---

## 五、爽点与憋屈的配比

### 憋爽比例公式

```
轻松期：爽7 : 憋3 （日常/甜蜜/发育）
压迫期：憋6 : 爽4 （被压制/困境）
爆发期：爽8 : 憋2 （逆袭/打脸/高潮）
```

### 憋屈阈值

| 类型 | 最大憋屈章数 | 超过后的风险 |
|-----|------------|------------|
| 小白文 | 3章 | 弃书 |
| 老白文 | 5章 | 弃书 |
| 虐文 | 10章 | 读者流失 |
| 甜文 | 1章没有糖 | 弃书 |

### 憋屈释放技巧

- 憋屈最多持续3章，第4章必须释放
- 释放时爽感要足够强，弥补前面的憋
- 可以用"小爽"过渡，避免读者流失

---

## 六、各题材爽点侧重点

| 题材 | 核心爽点 | 爽点密度 | 备注 |
|-----|---------|---------|------|
| 都市爽文 | 打脸/装逼/财富 | 高 | 节奏快，爽点密集 |
| 玄幻修仙 | 升级/打脸/传承 | 中 | 升级为主，节奏稍慢 |
| 甜宠文 | 吃醋/撒糖/护短 | 中 | 甜为主，打脸为辅 |
| 悬疑文 | 反转/揭露/智斗 | 低 | 爽点要精不要多 |
| 末世文 | 生存/囤货/异能 | 高 | 危机与爽并存 |

## 七、爽点金句模板

### 打脸金句

- "你说的这辈子......好像有点短？"
- "刚才谁说我是废物来着？"
- "不是我针对谁，在座各位都是垃圾。"

### 装逼金句

- "不是我想装，是你们逼我的。"
- "随便玩玩，没想到你们当真了。"
- "这玩意？我家孩子玩具。"

### 燃点金句

- "谁都不能动我在乎的人。"
- "此战，为兄弟！为宗门！"
- "燃尽一切，又如何！"

### 甜点金句

- "以后离他远点，听见没？"
- "我的人，只有我能欺负。"
- "想我了？嗯？"
