# 安全政策

本项目包含本地 Python 脚本、文档和 AI 技能说明。安全问题包括但不限于路径遍历、意外覆盖本地文件、命令注入、敏感信息泄露和供应链风险。

## 私密报告

仓库已启用 GitHub 私密漏洞报告。请使用 [Report a vulnerability](https://github.com/tinylion1024/web-novel-master/security/advisories/new) 提交可复现步骤、影响范围和建议修复方式；不要在公开 Issue 中披露可被直接利用的细节。

## 范围与响应

- 请使用最新 `main` 复现问题。
- 请勿提交用户小说正文、密钥、访问令牌或私人配置作为复现材料。
- 维护者会先确认问题，再在修复与披露方式确定后公开记录。
