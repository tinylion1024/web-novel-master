# Changelog

本项目遵循语义化版本记录用户可见变更。

## [1.1.1] - 2026-07-21

### Added

- 英文检索入口、快速演示与对外推广素材。
- 可用于 GitHub 社交预览的 1280x640 图片源。
- 三个面向新贡献者的 Issue 与首个入门 Discussion。

### Changed

- 仓库已启用 Template，并补充中文网文、连载创作和 AI Agent 相关 Topics。

## [1.1.0] - 2026-07-20

### Added

- 可复现的原创最小项目与四模式对照样例。
- Claude Code / OpenClaw 安装指南和可测试安装器。
- 平台定位、连续创作、数据来源核验模板。
- GitHub Discussions、内容与来源核验标签、Issue Forms、PR 模板和私密漏洞报告。
- 贡献、支持、发布和安全说明。

### Changed

- README 改为以首次成功、样例和安装路径为中心。
- 文档校验覆盖 `docs/` 与新增示例。

## [1.0.0] - 2026-07-17

### Added

- 四种网文创作模式、项目管理脚本、命名多样性检查与质量基线校验。
