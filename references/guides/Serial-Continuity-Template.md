# 连续创作模板

每次开始新章节前复制本模板。它把“上一章发生了什么”转成可检查的状态，适用于人工、子代理或团队续写。

## 本章输入

- 章节：第 [编号] 章
- 上一章一句话摘要：
- 读者已知信息：
- 作者保留信息：
- 本章必须兑现的承诺：
- 本章结尾的新问题：

## 章节交付检查

- [ ] 人名、地点、组织和道具已与 `03-写作计划.json` 的命名台账核对。
- [ ] 本章解决至少一个已建立的问题，或明确推进其代价。
- [ ] 新信息没有推翻前文既定事实；如需修订，已记录原因。
- [ ] 结尾留下可由下一章回应的具体问题。
