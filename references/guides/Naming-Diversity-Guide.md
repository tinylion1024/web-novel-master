# 命名多样化指南

> 适用范围：人物姓名、地名、组织名、功法名、法宝名、公司名、系统名。
> 目标：避免默认落入“林/顾/沈/苏/叶 + 县城/京城/魔都 + 某某集团/某某神功”的重复套路。

## 总原则

1. 先定气质，再定名字：命名要服务于题材、阶层、地域和人物气质。
2. 同项目内去重：主角、女主、反派、导师、核心城市、核心势力，尽量不要撞首字和整体音感。
3. 避免默认高频词：若用户未指定，不要把高频姓氏和泛化地点当第一反应默认值。
4. 一个项目至少有一层独特命名逻辑：可以是方言感、古意感、沿海工商感、边地粗粝感、宗门冷峻感。

## 人物姓名

### 不建议作为默认首选的高频姓氏

`林 / 顾 / 沈 / 苏 / 叶 / 秦 / 萧 / 楚 / 陆 / 傅 / 江 / 韩`

这些姓氏不是禁用，而是不要在用户未指定时作为第一反应默认值。

### 建议的姓氏分桶

- 都市现实向：`许 / 周 / 贺 / 程 / 唐 / 宋 / 梁 / 何 / 陶 / 魏 / 任 / 邵 / 谢 / 郝 / 方`
- 古言权谋向：`裴 / 崔 / 卢 / 温 / 谢 / 段 / 霍 / 殷 / 宁 / 晏 / 闻 / 宣`
- 玄幻仙侠向：`厉 / 姬 / 澹台 / 司空 / 闻人 / 左丘 / 申屠 / 百里 / 公仪 / 凌`
- 女频豪门向：`闻 / 温 / 容 / 盛 / 商 / 纪 / 岑 / 贺 / 宋 / 黎 / 施 / 俞`

### 自检

- 是否又落回了“林某某 / 顾某某 / 沈某某”？
- 主角和反派首字是否重复？
- 女主和主角是否都属于同一种审美词库？

## 地名

### 不要直接把泛称当正式地名

不推荐长期直接使用：

`县城 / 京城 / 魔都 / 小镇 / 大城市 / 边城 / 省城`

这些词可以作为功能描述，不应替代正式地名。

### 推荐公式

- 都市现实向：`自然地貌字 + 气候/色彩字 + 行政后缀`
- 玄幻仙侠向：`意象字 + 势力字 + 地貌字`

示例：

- 临岚市
- 霁川市
- 雾梁县
- 鹤汀县
- 断霞洲
- 青溟城

## 各种本体

### 公司/集团

避免反复使用：`天道集团 / 龙腾集团 / 鼎盛集团 / 华盛集团`

建议：

- 霁川医控
- 泊云重工
- 衡越生物
- 同栈资本

### 宗门/家族/帮派

避免全是：`天玄宗 / 万剑宗 / 青云门 / 叶家 / 林家`

建议：

- 烛海宫
- 北陵裴氏
- 听雪台
- 沉沙会

### 功法/法宝/系统

避免全是：`神级系统 / 无敌系统 / 九天神诀 / 混沌神剑`

建议：

- 折潮呼吸法
- 归藏针经
- 乌檀镇脉针
- 因果纠偏系统
