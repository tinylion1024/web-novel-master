**目标**：分析用户创作意向，匹配爆款网文市场趋势

---

根据用户初始输入，快速分析：

## 1.2 爆款套路速查

### 四大主流套路

| 套路 | 核心公式 | 适合赛道 |
|------|---------|---------|
| **退婚流** | 废物开局 → 当众羞辱 → 意外觉醒 → 打脸 | 都市/玄幻 |
| **系统流** | 濒死/危机 → 系统激活 → 任务奖励 → 变强 | 都市/玄幻 |
| **重生流** | 死亡/失败 → 重生节点 → 先知优势 → 复仇 | 历史/都市 |
| **装逼流** | 低调被逼 → 无奈展示 → 全场震惊 | 都市/玄幻 |

### 各赛道经典开局

**都市爽文**
```
退婚打脸：未婚妻当众退婚，主角当场反击
身份反差：穷小子原来是隐形富豪/兵王/神医
```

**玄幻修仙**
```
废柴逆袭：丹田破碎/灵根尽毁 → 偶得传承 → 惊艳崛起
宗门崛起：被逐出宗门 → 绝境中获得上古传承 → 打脸回归
```

**穿越重生**
```
历史穿越：穿成赘婿/废物 → 用现代知识降维打击
末世囤货：重生末世前 → 疯狂囤积资源 → 打造安全基地
```

## 1.3 提取用户意向

从用户输入中提取关键要素：

### 意向提取清单

| 要素 | 提取目标 | 记录字段 |
|------|---------|---------|
| 类型偏好 | 都市/玄幻/穿越/甜宠等 | `genre` |
| 金手指 | 系统/重生/异能/传承等 | `goldenFinger` |
| 核心爽点 | 打脸/升级/甜宠/装逼等 | `coreTropes` |
| 主角设定 | 男频/女频/双主角 | `protagonist` |

### 提取结果展示

如果信息充足，展示提取结果：

```markdown
## 已提取的创作意向

🎯 **类型**：[提取的类型]
🔮 **金手指**：[提取的金手指]
💥 **爽点**：[提取的核心爽点]
🧑 **主角**：[提取的主角设定]

确认以上信息，或进入详细问答完善设定。
```

## 1.4 快速推荐

根据提取的要素，给出个性化推荐：

### 推荐逻辑

- 用户提到"都市" + "打脸" → 推荐都市爽文，退婚流/身份反差流
- 用户提到"重生" + "复仇" → 推荐穿越重生流，历史穿越或都市重生
- 用户提到"修仙" + "升级" → 推荐玄幻修仙流，系统觉醒或传承流
- 用户提到"甜" + "宠" → 推荐甜宠言情，马甲掉落或病娇流

### 推荐话术模板

```
基于你的需求，我推荐以下创作方向：

【推荐1】[具体类型] — [核心卖点]
适合喜欢[用户提到的关键词]的读者，核心爽点是[爽点描述]。

【推荐2】[备选类型] — [备选卖点]
[备选描述]

【自定义】我有其他想法...
```
