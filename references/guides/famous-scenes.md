# 名场面设计指南

名场面是爆款网文的核心竞争力——让读者看到停不下来、忍不住分享的场景。

---

## 一、名场面的定义

**名场面 = 情绪爆点 × 戏剧张力 × 记忆锚点**

| 要素 | 说明 |
|-----|------|
| **情绪爆点** | 读者情绪被强烈调动（爽/哭/气/笑/燃） |
| **戏剧张力** | 有冲突、有反转、有意外 |
| **记忆锚点** | 有金句/名台词/名画面，读者能复述 |

---

## 二、名场面类型体系

### A. 爽点名场面（让读者"爽"）

| 类型 | 核心公式 | 典型场景 |
|-----|---------|---------|
| **打脸型** | 羞辱→反转→碾压 | 废柴当众出丑→露出真实身份→打脸全场 |
| **验证型** | 质疑→展示→震惊 | 众人不信→主角出手→满堂震惊 |
| **装逼型** | 低调→被逼→无奈展露 | 主角不想出手→被逼无奈→实力碾压 |
| **复仇型** | 隐忍→时机成熟→清算 | 多年隐忍→仇人落难→亲手复仇 |
| **逆袭型** | 绝境→金手指→翻盘 | 被逼到绝路→系统激活→绝地翻盘 |

### B. 情感名场面（让读者"哭/甜/虐"）

| 类型 | 核心公式 | 典型场景 |
|-----|---------|---------|
| **牺牲型** | 危机→抉择→牺牲 | 为了救XX，主角选择牺牲自己 |
| **告白型** | 误解→真相→告白 | 发现对方一直默默付出→真情告白 |
| **离别型** | 不得不走→不舍→承诺 | 不得不离开→含泪告别→下次相见 |
| **重逢型** | 多年未见→变化→珍惜 | 多年后重逢→物是人非→珍惜眼前 |
| **误会型** | 看到表面→真相→心痛 | 看到他和别的女人→其实是妹妹→后悔 |

### C. 燃点名场面（让读者"燃"）

| 类型 | 核心公式 | 典型场景 |
|-----|---------|---------|
| **热血型** | 不可能→拼命→奇迹 | 所有人说不行→主角拼死一搏→居然成功了 |
| **守护型** | 家人被威胁→爆发→碾压 | 家人被伤害→主角彻底愤怒→团灭对方 |
| **团战型** | 绝境→配合→翻盘 | 队友一个个倒下→最后逆转→团队胜利 |
| **觉醒型** | 压制→极限→觉醒 | 被压着打→触发条件→实力暴涨 |

### D. 悬疑/反转名场面（让读者"懵/惊"）

| 类型 | 核心公式 | 典型场景 |
|-----|---------|---------|
| **身份反转型** | 以为是好人→揭露→震惊 | 一直信任的人→居然是幕后黑手 |
| **实力反转型** | 以为很强→真相→打脸 | 以为主角是废物→其实是最强 |
| **关系反转型** | 以为是敌人→真相→惊呆 | 以为是竞争对手→其实是亲兄弟 |
| **阴谋揭示型** | 表面平静→揭露→大乱 | 看似风平浪静→揭露惊天阴谋 |

## 三、名场面时间轴规划

### 20章名场面节奏表

```
第1章：【觉醒名场面】系统激活/重生归来/退婚打脸
         → 让读者立刻上头

第3章：【首个打脸名场面】小试牛刀，初次验证爽点
         → 建立"这文很爽"的预期

第5章：【情感名场面】初次心动/误解/离别
         → 建立情感羁绊

第7章：【装逼名场面】当众打脸/实力展示
         → 积累"等主角碾压"的情绪

第10章：【第一个大高潮名场面】副本通关/复仇第一步
         → 第一个情绪高峰

第12章：【牺牲/虐心名场面】重要角色牺牲/离别
         → 制造情绪低谷，增强后续爽感

第15章：【身份反转名场面】揭露隐藏身份
         → 重大悬念释放

第18章：【团战/燃点名场面】守护XX/团队战斗
         → 情绪再拉高

第20章：【大结局名场面】终极对决/最终复仇/甜蜜结局
         → 高潮收尾
```

### 节奏原则

- 每3-5章必须有1个中型名场面
- 每10章有1个大型名场面（全书高潮锚点）
- 名场面之间要有"情绪缓冲"（日常/温馨/蓄力）

## 四、名场面设计三要素

### 1. 【蓄力】名场面之前必须有铺垫

羞辱要够狠（打脸才够爽）
误会要够深（解开才够甜）
压迫要够强（爆发才够燃）

**反例：**
> 主角突然就变强了，没有任何预兆

**正例：**
> 所有人都嘲笑他废物，未婚妻退婚，妹妹被欺负...第三章他出手时，读者已经憋了一肚子气等发泄

### 2. 【爆发】情绪到达顶点那一刻必须写透

动作/语言/心理描写要到位
不要在最高潮时收笔
金句/名台词必须出现

**反例：**
> 他展示了实力，所有人都震惊了。

**正例：**
> 他轻轻抬手。\n\n"且慢。"\n\n声音不大，却像一颗石子投入死水，整个大殿瞬间安静。\n\n那道笼罩在他身上的封印，碎了。\n\n"你说什么？"大宗师的声音有些发颤。\n\n"我说，"他抬起头，眼底是三年来压抑的怒火，"你们欠我的，该还了。"

### 3. 【余韵】名场面后要有情绪延续

描写周围人的反应
描写主角的心理状态
为下一个名场面埋下种子

## 五、名场面速写公式

### 打脸型名场面公式

```
【蓄力】XX羞辱主角，说"你这辈子也不可能..."
【爆发】主角微微一笑，露出/说出/做出XX
【结果】全场震惊，XX脸色惨白
【金句】"你说的这辈子...好像有点短？"
```

**完整示例：**
> "废物就是废物，"王少爷把茶泼在他脸上，"这辈子你就只配捡本少爷的剩饭。"
>
> 林墨没有动。他只是笑了笑。
>
> 门外，十几辆黑色迈巴赫缓缓停下。一位白发老者下车，快步走进酒店。
>
> "家主！"老者在他面前单膝跪地，"属下接驾来迟！"
>
> 王少爷的脸，绿了。

### 装逼型名场面公式

```
【背景】众人轻视，主角被无视
【挑衅】有人故意刁难/羞辱
【无奈】主角叹气："既然你们想看..."
【展示】主角展示实力/说出真相
【金句】"不是我想装，是你们逼我的。"
```

### 燃点型名场面公式

```
【绝境】队友倒下，敌人逼近，资源耗尽
【触发】XX（家人/爱人/信念）被触碰
【爆发】主角觉醒/突破/爆发
【碾压】敌人从嘲笑变成恐惧
【金句】"谁都不能动我在乎的人。"
```

### 甜宠型名场面公式

```
【吃醋】看到他/她和别人在一起
【行动】直接拉过/壁咚/宣示主权
【金句】"以后离他远点，听见没？"
【反应】对方心跳加速/脸红/挣扎
```

## 七、金句库

### 打脸金句

- "你说的这辈子...好像有点短？"
- "不是我针对谁，在座各位都是垃圾。"
- "刚才谁说我是废物来着？"
- "这世上有些东西，你们永远够不着。"

### 装逼金句

- "不是我想装，是你们逼我的。"
- "我只是随便玩玩，没想到你们当真了。"
- "这东西？我家小孩的玩具罢了。"

### 燃点金句

- "谁都不能动我在乎的人。"
- "这一战，为宗门！为兄弟！为信念！"
- "燃尽一切，又如何！"

### 甜宠金句

- "以后离他远点，听见没？"
- "我的人，只有我能欺负。"
- "想我了？嗯？"

## 八、名场面密度检查

### 每2000字必须包含

| 类型 | 数量 | 说明 |
|-----|------|------|
| 小爽点 | 1-2个 | 轻微打脸/小惊喜/暧昧互动 |
| 中爽点 | 1个 | 明确打脸/情感升温/热血时刻 |
| 钩子 | 1-2个 | 信息炸弹/悬念/下章预告 |

### 检查清单

- [ ] 本章有爽点吗？（如果没有，读者会感到"水"）
- [ ] 爽点之前有蓄力吗？（没有蓄力的打脸不够爽）
- [ ] 有金句吗？（没有金句的名场面不够"名"）
- [ ] 结尾钩子够强吗？（能让读者想点下一章吗）
