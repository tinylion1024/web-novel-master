# Phase 8 – Hook & Packaging

**目标**：优化标题、包装章节、规划连载钩子

---

## 8.1 章节标题优化

### 爆款标题公式

**都市爽文**
```
[身份] + [反差] + [爽点]
示例：上门女婿是神医 / 兵王回归，发现女儿住狗窝
```

**玄幻修仙**
```
[境界] + [突破] + [爽点]
示例：废物逆袭成武帝 / 丹田破碎后，我觉醒混沌体
```

**穿越重生**
```
[身份] + [逆袭] + [爽点]
示例：重生1998，商界教父 / 穿越成赘婿，我靠系统躺赢
```

**甜宠言情**
```
[身份] + [反差] + [甜点]
示例：总裁夫人马甲掉了 / 影帝的小糊咖突然爆红了
```

### 标题检查清单

- [ ] 标题是否包含核心爽点？
- [ ] 是否有身份反差/悬念？
- [ ] 是否能让读者产生好奇？
- [ ] 是否不超过20字？

### 标题优化示例

```
原标题：第1章 修炼
优化：第1章 废物？金丹期的护体灵气碎了

原标题：第5章 相遇
优化：第5章 民政局门口，她等了他三小时
```

## 8.2 章节简介优化

### 每章简介公式

```
[本章核心爽点] + [悬念钩子]
示例：
他当着全场的面，轻轻一弹。
金丹期护体灵气，碎了。
未婚妻脸色惨白——
而这，才刚刚开始。
```

### 简介检查

- [ ] 是否在50字以内？
- [ ] 是否包含本章最大爽点？
- [ ] 是否留下悬念？
- [ ] 是否有让读者想点进来的冲动？

## 8.3 连载钩子规划

### 钩子类型

| 类型 | 效果 | 示例 |
|------|------|------|
| **信息炸弹** | 揭示惊人信息 | "爸，你还活着？" |
| **冲突升级** | 矛盾即将爆发 | "三辆奔驰拦住去路" |
| **悬念收尾** | 留下未解之谜 | "手机响了，屏幕显示：爸爸" |
| **爽点预告** | 暗示下章更爽 | "他不知道，三天后…" |

### 连载钩子分布

```
第1章结尾：最强爽点预告
第2-3章结尾：日常钩子
第4章结尾：大钩子（名场面后）
第5章结尾：悬念钩子
循环...
```

## 8.4 作品包装

### 书名优化

```
爆款书名 = [核心爽点] + [身份/反差] + [悬念]
示例：
- 《都市：我真不是废物》（打脸爽）
- 《修仙：开局获得上古传承》（系统爽）
- 《甜宠：影帝的小糊咖马甲掉了》（甜宠爽）
```

### 简介撰写

```
公式：[主角身份] + [困境] + [金手指] + [爽点预告]

示例：
窝囊上门女婿陈昊，被丈母娘当众羞辱。
就在所有人以为他完了的时候——
他手机响了，来电显示：妈，我是爸。
（他是隐世家族遗孤，而那个家族，富可敌国...）
```

### 标签设定

```
核心标签：都市 / 玄幻 / 甜宠 等
爽点标签：打脸 / 升级 / 甜宠 等
卖点标签：神医 / 兵王 / 穿越 等
```
