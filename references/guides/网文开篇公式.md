# 网文开篇公式

网文开篇决定读者是否继续看下去。前200字必须抓住读者，前3000字必须让读者上头。

---

## 一、开篇核心原则

### 黄金前三段

```
第一段：必须出现冲突/危机/悬念
第二段：必须让读者想知道"接下来会怎样"
第三段：必须建立主角让读者想跟随
```

### 开篇三大禁忌

| 禁忌 | 反例 | 问题 |
|-----|------|------|
| 环境描写 | "那天天气晴朗..." | 读者无感，无代入 |
| 日常流水 | "主角起床刷牙洗脸..." | 没有冲突，弃书 |
| 背景说明 | "这个世界有三种职业..." | 信息轰炸，枯燥 |

### 开篇公式

```
【最佳开篇】
行动中开场 → 即时冲突 → 悬念钩子

【次佳开篇】
危机时刻 → 主角应对 → 展示金手指/实力
```

## 二、各类型开篇公式

### 都市爽文

#### 神医流

```
【公式】
困境 + 身份反差 + 展示实力

【模板】
主角身份：表面废物/实习/被看不起
        ↓
出现危机：有人生命垂危无人能救
        ↓
主角出手：三针/一药/一句话
        ↓
震惊全场：专家/领导/所有人震惊
```

**示例：**
> "血压还在降，准备后事吧。"
>
> 家属哭成一片。实习生林墨挤进人群，掏出银针。
>
> "你干什么！"主任呵斥，"一个实习中医也敢——"
>
> 三针落下。监护仪的直线，变成规律的波浪。
>
> 主任的脸，僵住了。

#### 兵王流

```
【公式】
废物身份 + 故人求助 + 身份反差

【模板】
归隐背景：退役战神/被误解/受委屈
        ↓
故人出现：妻儿被欺负/家人有难
        ↓
霸气回应：一个电话/一个动作
        ↓
身份揭露：首长敬礼/全场震惊
```

**示例：**
> "你这个废物，连自己老婆都保护不了！"
>
> 老丈人把茶杯砸在他脸上。
>
> 门口，十几辆军用吉普停下。一位肩扛三星的上将下车，快步走来。
>
> "首长！您真的在这里！"
>
> 将军立正敬礼。废物女婿，缓缓起身。
>
> 老丈人的腿，软了。

#### 总裁流

```
【公式】
契约/隐婚 + 当众羞辱 + 反转

【模板】
隐婚状态：男主隐藏身份/被当废物
        ↓
当众羞辱：未婚妻/情敌嘲讽
        ↓
女主出现：维护男主/亮出身份
        ↓
打脸时刻：他才是真正大佬
```

**示例：**
> "三千万，买你老婆一夜，够不够？"
>
> 情敌把支票甩在桌上。
>
> 沈默拿起酒杯："钱留着看病吧。"
>
> 女人突然笑了："老公，别逗了。"
>
> 她亮出手机："刚收到消息，您家族那百亿项目......审批完了。"
>
> 情敌的脸，绿了。

### 玄幻修仙

#### 废材逆袭流

```
【公式】
废物开局 + 当众羞辱 + 意外觉醒

【模板】
丹田破碎：修炼废物，受尽嘲笑
        ↓
当众羞辱：退婚/测灵力/比试
        ↓
意外获得：戒指/传承/系统/记忆
        ↓
实力展现：震惊全场
```

**示例：**
> "林墨，丹田破碎，终生无法修炼。"
>
> 长老的声音回荡在演武场。嘲笑声四起。
>
> 未婚妻走上前，当众退还婚书："废物不配进我赵家门。"
>
> 林墨接过婚书，笑了。
>
> 脑海中，沉睡十年的青铜古灯，亮了。

#### 系统觉醒流

```
【公式】
死亡/危机 + 系统激活 + 新手任务

【模板】
濒死状态：被围攻/重伤/绝望
        ↓
系统激活：叮咚声/声音提示
        ↓
获得礼包：新手礼包/新手任务
        ↓
立刻变强：任务奖励/实力提升
```

**示例：**
> 刀光闪过。他最后的念头是：这就死了？
>
> "叮！检测到宿主死亡，系统激活中......"
>
> "叮！宿主灵魂绑定成功，获得新手大礼包！"
>
> "叮！检测到宿主重生是否激活？"
>
> 林墨睁开眼。回到了十年前，一切悲剧发生的前一天。

### 甜宠文

#### 契约婚姻流

```
【公式】
契约结婚 + 隐婚曝光 + 真香

【模板】
契约状态：隐婚/各过各的/互不干涉
        ↓
相处日常：甜宠互动/互相照顾
        ↓
危机曝光：被家人/同事/情敌发现
        ↓
真香时刻：老公/老婆原来是TA
```

**示例：**
> "我们只是契约关系，别越界。"
>
> 顾太太递来协议。
>
> 陆霆之接过，大笔一挥签下名字。心里却在想：早晚让你收回这句话。
>
> 三个月后。
>
> "老公，他们说我被潜规则了......"
>
> 某人放下手机，眼底发凉："哪个不长眼的？"

#### 马甲掉落流

```
【公式】
多重身份 + 身份互补 + 陆续揭露

【模板】
表面身份：普通/废物/被看不起
        ↓
隐藏身份：各领域大佬/隐藏实力
        ↓
陆续揭露：一个个马甲被揭开
        ↓
最终揭示：所有人知道真相
```

**示例：**
> 她是被婆家赶出门的废物媳妇。
>
> 某天，一份DNA报告震惊全城——她是失散首富家的亲生女儿。
>
> 还没完。
>
> 某跨国集团继承人："她是我们集团最大股东。"
>
> 某秘密组织长老："她是我们组织的创始人。"
>
> 某人将她护在身后："她是我老婆，谁也别想抢。"

### 穿越重生

#### 历史穿越流

```
【公式】
死亡/意外 + 穿越古代 + 先知优势

【模板】
穿越前：现代人/知道历史/有技能
        ↓
穿越点：落难/废物/被欺负
        ↓
利用优势：用现代知识/预知历史
        ↓
改变命运：发家致富/抱大腿/打脸
```

**示例：**
> 李默睁开眼睛，看到的是茅草屋顶和破旧的棉被。
>
> 记忆涌入——这是个历史上不存在的朝代。
>
> 门外传来哭声："儿啊，爹娘对不起你，没钱给你治病......"
>
> 李默坐起身。不存在的朝代？那历史走向，岂不是......由他说了算？

#### 末世囤货流

```
【公式】
重生末世 + 知道末日日期 + 提前准备

【模板】
重生节点：回到末世前一周/一月
        ↓
知道信息：末世爆发时间/持续多久
        ↓
囤货建设：物资/武器/基地/异能
        ↓
危机应对：末世降临后的生存
```

**示例：**
> "还有三天，丧尸病毒就会爆发。"
>
> 周明重生回来，第一件事是请假。
>
> 第二件事是——清空所有银行卡，冲进超市。
>
> 食物、水、药品、工具......够支撑一年。
>
> 第三天，闹钟响起。世界，彻底变了。

## 三、开篇钩子类型

### 信息炸弹型

开篇抛出震撼信息，读者必须想知道原因。

```
"我亲手杀了我最好的朋友。"
        ↓
"三天前。"
```

### 悬念型

开篇提出问题，读者想知道答案。

```
她手机里有一条未读消息。
发送时间是——她死后。
```

### 冲突型

开篇就是冲突，读者想知道结果。

```
刀架在脖子上。
"叫一声，就杀了你妈。"
```

### 金句型

开篇一句让读者震惊的话。

```
"我老婆的孩子，不是我的。"
```

## 四、开篇检查清单

### 必须有

- [ ] 冲突或危机
- [ ] 主角让读者想跟随
- [ ] 钩子让读者想继续

### 绝对不能有

- [ ] 环境描写（天气、风景）
- [ ] 主角起床/洗漱/吃饭
- [ ] 大段背景设定
- [ ] 无意义的寒暄对话

### 黄金前三段检查

- [ ] 第一段有冲突/危机吗？
- [ ] 第二段让读者想知道"然后呢"吗？
- [ ] 第三段主角做了什么让读者想跟？

---

## 五、第1章特殊要求

第1章是读者决定"是否追这本书"的关键。

### 必须完成的任务

1. **建立爽文基调**：让读者知道"这文很爽"
2. **建立主角人设**：让读者想跟随主角
3. **留下大钩子**：让读者想看第2章

### 第1章爽点公式

```
第1章 = 觉醒爽点 + 大钩子

觉醒爽点：系统激活/退婚打脸/身份揭露
        ↓
大钩子：下一章会发生什么的悬念
```

### 失败案例

```
第1章写了主角日常生活
→ 读者：无聊，弃书

第1章写了几千字世界观设定
→ 读者：太枯燥，弃书

第1章没有爽点
→ 读者：没意思，弃书
```
