# 审核目录

审核清单目录 - 自检清单、避坑指南、质量审核。

## 目录结构

```
Check/
├── Pre-Writing/           # 写作前检查
│   ├── Idea-Validation.md   # 创意可行性验证
│   ├── Market-Fit.md        # 市场契合度检查
│   └── Outline-Review.md     # 大纲审核
├── During-Writing/        # 写作中检查
│   ├── Chapter-Check.md      # 章节自检
│   ├── Consistency-Check.md # 一致性检查
│   └── Pacing-Check.md      # 节奏检查
├── Post-Writing/          # 写作后检查
│   ├── Quality-Review.md    # 质量审核
│   ├── Beta-Reader.md       # 读者反馈处理
│   └── Final-Polish.md      # 最终润色检查
└── Pitfalls/              # 避坑指南
    ├── Plot-Pitfalls.md     # 剧情避坑
    ├── Character-Pitfalls.md # 人设避坑
    └── Writing-Pitfalls.md  # 写作避坑
```
