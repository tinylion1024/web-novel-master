# Phase 5 – Outline Planning

**目标**：生成完整大纲、人物档案、写作计划

---

## 5.1 创建项目文件夹

`./web-novels/{YYYYMMDD-HHmmss}-{小说名称}/`

## 5.5 规划摘要展示 + 写作模式选择

```markdown
## 《[小说名称]》创作规划确认

📊 **规模**: [X] 章 | 每章 2000-3000 字 | 预计总字数 [X-X] 万字

🧑 **主角**: [主角名称] | [身份] | [性格]
👥 **女主**: [女主名称] | [类型]
😈 **反派**: [反派类型] | [特点]

⭐ **名场面**: [X] 个（每 X 章 1 个）
- 第1章: [名场面类型] - [描述]
- 第3章: [名场面类型] - [描述]
- ...

🔥 **核心爽点**: [核心爽点]
🔮 **金手指**: [金手指]

---
**请选择写作模式：**
- 【串行】主 Agent 逐章写，稳定可靠（适合短中篇）⭐
- 【并行】子 Agent 分批并行，追求速度（适合中长篇）
- 【Teams】Claude Code 多 Agent 协作（适合大型长篇，需手动开启）
```

```
Question: 确认以上规划并选择写作模式
Options:
- 确认，开始创作（串行模式）⭐
- 确认，使用并行模式
- 确认，使用 Agent Teams 模式
- 我想修改某些设置
```

**用户确认后**：
- 更新 `03-写作计划.json` 的 `writingMode` 字段
- `status` 设为 `"in_progress"`
- → 进入 **Phase 6: Draft Writing**
