# Soul Directory

文风参考目录 - 不同赛道、读者群体、作者风格的参考素材。

## 目录结构

```
Soul/
├── Male-Oriented/          # 男频文风参考
│   ├── Urban-Action.md    # 都市爽文风格
│   ├── Xianxia.md         # 玄幻修仙风格
│   └── Gaming.md          # 游戏异界风格
├── Female-Oriented/        # 女频文风参考
│   ├── Romance.md         # 言情甜宠风格
│   └── Fantasy.md         # 女强玄幻风格
├── Category/              # 赛道风格参考
│   ├── Quick-Read.md     # 快节奏冲榜风格
│   ├── Long-Arc.md       # 长篇运营风格
│   └── Serial.md         # 连载追读风格
└── Author-Style/          # 作者风格参考
    ├── Compact.md         # 简洁干练风格
    ├── Descriptive.md     # 细腻描写风格
    └── Humorous.md       # 幽默风趣风格
```
