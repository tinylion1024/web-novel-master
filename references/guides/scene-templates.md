# 名场面模板库

各题材爆款网文名场面速写模板，可直接套用。

---

## 一、都市爽文名场面模板

### 1. 神医名场面

**场景：濒死患者 → 无人能救 → 主角出手 → 震惊全场**

```
【前提】患者生命垂危，西医宣告无效，家属崩溃

【阻碍】主任专家阻止："你一个实习生凭什么..."

【打脸】主角三针下去/一剂药/一句话
        ↓
患者情况好转/痊愈
        ↓
主任："这不可能！"
        ↓
专家/领导/院长震惊
        ↓
主角被追问来历

【金句】
"西医做不到的事，中医可以。"
"三年不起眼的实习生，今天让你们开开眼。"
"不是我看不起西医，是你们确实差点意思。"
```

### 2. 兵王名场面

**场景：废物女婿被羞辱 → 故人来接 → 身份揭露 → 全场跪**

```
【前提】丈人/妻子被嘲讽，主角被当废物

【冲突】"你这个废物，连老婆都保护不了！"

【反转】门口军用车辆停下
        将军/首长/老战友出现
        ↓
"首长！属下接驾来迟！"
        ↓
将军向废物女婿敬礼
        ↓
全场死寂

【金句】
"我只想安静当个废物，你们偏不让我如愿。"
"曾经守护过的东西，没人能动。"
"首长？哦，那个时代的事了。"
```

### 3. 总裁名场面

**场景：契约婚姻 → 被当众羞辱 → 身份曝光 → 打脸**

```
【前提】女主/男主隐藏身份，被当穷酸/废物

【冲突】情敌甩支票/当众退婚/嘲讽没钱
        ↓
"三千万，买她一夜够不够？"
        ↓
主角拿起支票，笑了

【反转】女主/男主亮出身份
        ↓
"XX集团老板/最大股东/实际控制人"
        ↓
情敌家族需要求着主角
        ↓
主角淡淡："钱留着治病吧。"

【金句】
"我从来没说过我没钱，是你以为我没钱。"
"契约只是保护她，不是保护你们。"
"想买我老婆一夜？你的钱够买你全家的命吗？"
```

### 4. 保镖名场面

**场景：雇主被威胁 → 主角出手 → 一个打十个 → 震惊**

```
【前提】雇主被绑架/被威胁/陷入危机
        ↓
保镖/保安被轻松打倒
        ↓
"没人能救你了。"

【爆发】主角出手
        ↓
一个打十个/打二十个/打一百个
        ↓
动作干净利落，毫不费力
        ↓
敌人从嚣张到恐惧

【金句】
"我收的工资高，是因为我值这个价。"
"动我保护的人？给你们三秒钟跑。"
"放心，我下手有分寸，死不了。"
```

## 二、玄幻修仙名场面模板

### 1. 废材逆袭名场面

**场景：丹田破碎 → 当众退婚 → 意外觉醒 → 震惊全宗**

```
【憋屈】丹田测试：废物，不配当修士
        ↓
退婚现场：未婚妻当众退还婚书
        ↓
"废物不配进我赵家门。"
        ↓
全宗嘲笑

【爆发】主角接过婚书，笑了
        ↓
脑海中：沉睡的传承觉醒/戒指老爷爷/系统激活
        ↓
一股气势从主角身上爆发
        ↓
测试石：不是废物，是顶级天赋！
        ↓
长老/宗主震惊

【金句】
"三十河东，三十年河西，莫欺少年穷。"
"你们说我废物？那今天就让你们看看什么叫天才。"
"婚书我收了，但——是我休的你。"
```

### 2. 炼丹名场面

**场景：被嘲笑不会炼丹 → 炼制神丹 → 丹雷降临 → 打脸**

```
【憋屈】炼丹测试：成功率为零
        ↓
被嘲笑："连炼丹都不会，还想进内门？"
        ↓
未婚妻："跟他解除婚约，他不配。"

【爆发】主角拿出药材
        ↓
开始炼丹，动作行云流水
        ↓
成丹！品质：神品！
        ↓
天空变色——丹雷降临！
        ↓
"这是......神丹丹雷？！"
        ↓
全场跪

【金句】
"不是不会炼，是你们不配看。"
"我这丹，能活死人肉白骨。"
"炼丹？不好意思，我炼的是天劫。"
```

### 3. 宗门大比名场面

**场景：垫底弟子 → 一路碾压 → 宗主震惊 → 身份成谜**

```
【憋屈】宗门大比，所有人看不起主角
        ↓
"林墨？那个废物？第一轮都过不了。"
        ↓
抽签对手嘲笑

【爆发】第一轮：主角轻松获胜
        ↓
第二轮：碾压种子选手
        ↓
八强：击败第一天才
        ↓
四强：击败宗主亲传弟子
        ↓
决赛：对手弃权
        ↓
全场不敢置信

【金句】
"你们说我废物？那今天就让你们看看。"
"不是我想低调，是你们太弱了。"
"这场比赛，我只是随便打打。"
```

### 4. 传承觉醒名场面

**场景：接受传承 → 考验通过 → 获得力量 → 众人跪**

```
【考验】传承之地开启
        ↓
众弟子尝试，全部失败
        ↓
"这传承千年无人能完整获得。"

【爆发】主角踏入
        ↓
一关、两关、三关......全部通过
        ↓
传承神殿发光——接受传承！
        ↓
远古意志："多少年了，终于有人完整接受。"
        ↓
主角获得的力量震惊长老
        ↓
宗主亲自出迎

【金句】
"我等这个传承，等了很久。"
"不好意思，这传承是我的了。"
"你们拜的祖师，是我故人。"
```

## 三、甜宠文名场面模板

### 1. 吃醋名场面

**场景：看到她/他和别人在一起 → 醋王上线 → 壁咚/宣示主权**

```
【吃醋触发】看到对象和异性单独在一起/被搭讪/笑容很甜
        ↓
内心翻涌，表面平静
        ↓
走向对方

【行动】一把拉过/壁咚/揽入怀中
        ↓
看着对方/看着那个异性
        ↓
"他/她是谁？"

【宣示主权】"以后离他/她远点，听见没？"
        ↓
霸道宣言/吻下去/公主抱
        ↓
对方脸红心跳

【金句】
"我没说你可以离开我视线。"
"吃醋？对自己老婆吃醋有错吗？"
"记住了，你是我的。"
```

### 2. 护短名场面

**场景：老婆/老公被欺负 → 霸气护短 → 打脸对方**

```
【冲突】老婆/老公被嘲讽/被刁难/被打压
        ↓
主角出现
        ↓
"就是你们欺负我老婆？"

【护短】对方嘲笑主角
        ↓
主角一个眼神/一个动作
        ↓
对方被震慑/被教训
        ↓
"敢动我的人，你是第一个。"

【金句】
"她是我老婆，我的人，谁都不能欺负。"
"道歉，不然你们公司明天就没了。"
"我从不惹事，但我的人被欺负了，这事就大了。"
```

### 3. 马甲掉落名场面

**场景：多个马甲陆续揭露 → 身份层层震惊 → 甜**

```
【表面身份】废物/普通人/被看不起

【揭露第一个马甲】
女主是某集团继承人
        ↓
"她是我们失散多年的女儿？"

【揭露第二个马甲】
她是某秘密组织长老
        ↓
"什么？她居然是...？！"

【揭露第三个马甲】
她还是某隐世家族传人
        ↓
"不可能！这怎么可能！"

【男主反应】
一直在旁边淡定喝茶
        ↓
"老婆，还有什么身份没说的？"

【金句】
"老婆，你的马甲还有几个？"
"我以为我娶了个普通人，原来是捡到宝了。"
"全城人都震惊了，只有他......好像早就知道？"
```

### 4. 霸道宣言名场面

**场景：被人看不起这段感情 → 男/女主霸气回应**

```
【冲突】有人质疑这段感情
        ↓
"你/她这种身份，怎么配得上他/我？"
        ↓
沉默片刻

【爆发】直视对方
        ↓
"他/她是我的男/女朋友，配不配，我自己说了算。"
        ↓
霸道拉过对象
        ↓
"以后再让我听到这种话，后果自负。"

【金句】
"就算全世界反对，我也要和他/她在一起。"
"你可以不喜欢我，但请不要质疑我的眼光。"
"我爱他/她，与你何干？"
```

## 四、悬疑反转名场面模板

### 1. 身份反转型

**场景：以为是好人 → 揭露是幕后黑手 → 震惊**

```
【表面身份】慈祥长辈/值得信任的导师/亲密朋友
        ↓
调查深入
        ↓
发现蛛丝马迹

【反转揭露】
他就是幕后黑手！
        ↓
一切都是他的布局
        ↓
"原来从一开始，就是你。"

【金句】
"谢谢你相信我，可惜......你信错了人。"
"我等这一天，等了很久。"
"你以为你在调查我，其实你一直在我的棋盘上。"
```

### 2. 实力反转型

**场景：以为主角是废物 → 揭露最强身份 → 打脸**

```
【表面】废物/没背景/没实力
        ↓
被打压/被嘲笑

【反转】关键时刻出手
        ↓
展示实力：最强/最高/最神秘
        ↓
"你们以为我是废物？"

【金句】
"不好意思，我只想安静当个废物。"
"你们够不着的地方，恰好是我的起点。"
"让你们失望了，我不是什么废物——我是你们惹不起的人。"
```

### 3. 关系反转型

**场景：以为是敌人 → 揭露是亲人/盟友 → 震惊**

```
【表面关系】竞争对手/仇家/陌生人
        ↓
真相揭露
        ↓
他们是亲兄弟/亲姐妹/亲生父母
        ↓
或者：他的爸爸是杀我爸爸的凶手
        ↓
等等——他爸爸是被陷害的！真凶另有其人！

【金句】
"我们以为的仇人，原来是血脉至亲。"
"命运开了一个玩笑，但我们终究是一家人。"
"有些真相，需要用一生去寻找。"
```

## 五、通用名场面模板

### 1. 装逼型通用模板

```
【前提】被看不起/被嘲讽/被打压
        ↓
被逼出手/被迫证明
        ↓
叹气："既然你们想看..."
        ↓
随手展示/简单操作
        ↓
震惊全场
        ↓
"不是我想装，是你们逼我的。"

【金句】
"随便玩玩而已，别当真。"
"我本来想低调，奈何实力不允许。"
"这点小事还要我亲自出手？"
```

### 2. 打脸型通用模板

```
【憋屈】被羞辱，被踩在脚下
        ↓
忍，沉默，隐忍
        ↓
时机成熟
        ↓
亮出身份/实力/底牌
        ↓
"你刚才说什么来着？再说一遍？"

【金句】
"我当时没说话，不是因为我没底气——是因为懒得跟你们计较。"
"今天这个脸，我本来不想打。"
"现在知道错哪儿了吗？"
```

### 3. 燃点型通用模板

```
【绝境】队友倒下/敌人太强/无法战胜
        ↓
信念触发：为了谁/为了什么
        ↓
"你们动了我在乎的人/东西。"
        ↓
爆发！突破！觉醒！
        ↓
碾压！
        ↓
"我说过，谁都不能动我在乎的人。"

【金句】
"这一战，为兄弟！为宗门！为信念！"
"燃尽一切，又如何！"
"你们惹错人了。"
```
