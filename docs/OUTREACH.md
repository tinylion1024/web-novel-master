## English Short Post

Web Novel Master is an open-source Claude Code / OpenClaw skill for structured Chinese web-fiction projects. It scaffolds original stories into outlines, character records, chapter plans, naming ledgers, and continuity checks, with local Python validation and reproducible examples. It is a workflow, not a popularity guarantee:

https://github.com/tinylion1024/web-novel-master
