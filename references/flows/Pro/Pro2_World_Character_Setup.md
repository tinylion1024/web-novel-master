## Q6: 情感线

```
请选择情感线（输入数字）：

【1】单线 — 1V1，专一
【2】多线 — 1V2或1V3
【3】无CP — 无感情戏
```

## Q7: 反派设计

```
请选择反派类型（输入数字）：

【1】单一强敌 — 从头到尾一个主要反派
【2】阶段BOSS — 每个阶段一个主要敌人
【3】内部背叛 — 身边人/亲人/兄弟反目
【4】势力BOSS — 家族/宗门/公司对抗
【5】幕后黑手 — 最终BOSS隐藏很深
【6】自由描述 — 告诉我你的想法
```

**追问 — 反派特点**：
仗势欺人 / 看不起主角 / 抢资源 / 背后捅刀 / 自以为是

**记录**：`antagonistType`、`antagonistTraits`
