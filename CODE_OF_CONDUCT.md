# 行为准则

本项目希望形成一个对创作者、读者和贡献者都友好的协作环境。

## 期待的行为

- 讨论作品方法、流程和证据时保持尊重、具体和建设性。
- 尊重不同题材、创作目标和使用的工具。
- 接受有依据的反馈，并清楚区分个人偏好与项目事实。
- 保护他人的未公开作品、个人信息和署名权。

## 不可接受的行为

- 人身攻击、歧视、骚扰或恶意贬损。
- 未经许可发布他人的文本、私人信息或付费内容。
- 以复刻在世作者可辨识表达为目标提交内容。
- 垃圾信息、恶意链接或与项目无关的推广。

## 处理方式

请通过仓库 Issue 或维护者可见的渠道报告问题。维护者会基于上下文采取提醒、编辑、删除内容或限制互动等措施。
